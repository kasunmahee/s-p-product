# POS Mini - Shop Supply Management System

A lightweight, mobile-responsive Shop Supply Management System built with HTML, Tailwind CSS, and Dexie.js (IndexedDB). No server or installation required.

## How to Run
Simply double-click `index.html` to open it in your web browser (Chrome, Edge, Firefox, Safari).

## Features
- **Dashboard**: View sales overview and recent transactions.
- **Shops Management**: Add, Edit, and View customer shops.
- **Inventory**: Manage products, cost prices, and selling prices.
- **Billing**: Create new supply bills with a cart system.
- **History**: View past transaction history with search.
- **Offline First**: All data is stored locally in your browser using IndexedDB.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Dexie.js (CDN)
- Lucide Icons (CDN)
- JavaScript (ES6+)

## Mobile Responsive
Designed with a bottom navigation bar for a native app-like experience on mobile devices.
